# bitkub-contracts

A community edition library for developing smart contracts on bitkub chain

### Installation

Install via `npm`
``` shell
npm install --dev @masdxi/bitkub-contracts
```
Install via `yarn`
``` shell
yarn add --dev @masdxi/bitkub-contracts
```
### Usage

``` solidity
import "@masdxi/bitkub-contracts/<path>/<file>.sol";
```
