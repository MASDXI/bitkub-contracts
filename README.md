# bitkub-contracts

A library for developing smart contracts on bitkub chain

### Installation

Install via `npm`
``` shell
npm install --dev @masdxi/bitkub-contracts@stable
```
Install via `yarn`
``` shell
yarn add --dev @masdxi/bitkub-contracts@stable
```
